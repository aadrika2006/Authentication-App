Full-Stack Authorization App (Node + React + SQLite)

This repository provides a complete, working authorization system using:

Backend: Node.js + Express + SQLite (via better-sqlite3)

Frontend: React (with Vite)

Auth: JWT Access Tokens + Refresh Tokens stored in HttpOnly cookies

Security: Bcrypt password hashing, token rotation, and CORS protection
